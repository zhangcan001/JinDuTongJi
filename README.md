# JinDuTongJi

监理工程师专用的工程进度管控台，用于查看楼栋、楼层、专业单位和施工内容的完成情况。界面采用太空舱中控大屏风格，支持 3D 沙盘、进度预警、Excel 导入和监理周报生成。

## 主要功能

- 中控大屏：展示总体完成率、关键节点、监理指令和项目覆盖范围。
- 3D 楼栋沙盘：按楼栋和楼层显示完成率，点击楼层查看具体施工情况。
- 楼层明细：展示单位、施工内容、进度、计划日期、实际日期和监理意见。
- 专业筛选：支持按楼栋、专业单位、施工内容和状态筛选模型数据。
- 进度预警：自动识别滞后、临期、已完成和施工中节点。
- Excel 导入：通过进度表更新全局数据，并校验缺失字段或异常内容。
- 地下室剖面：单独展示地下室各系统进度。
- 监理周报：根据当前数据自动生成周报文本。

## 项目范围

项目内置以下施工范围作为演示数据：

- 机电单位：室内给水系统、热水系统、室内排水系统、建筑中水系统、水系统末端设备安装、空调风管安装、空调设备安装。
- 消防单位：喷淋系统、消火栓系统、自动跟踪灭火系统、水系统末端设备安装、桥架敷设、电缆敷设、导管内穿线、电气末端安装、防排烟风管安装。
- 智能化单位：智能化桥架敷设、智能化线缆敷设、导管内穿线、智能化末端设备安装。
- 电梯单位：设备安装。

## 本地运行

这是一个静态前端项目，不需要安装依赖。可以直接打开 `index.html`，或用本地静态服务运行：

```powershell
npx serve .
```

如果当前端口是 `4173`，浏览器访问：

```text
http://127.0.0.1:4173/
```

## Excel 导入字段

建议进度表包含以下列：

- 项目
- 施工部位
- 楼层
- 专业
- 责任单位
- 施工内容
- 节点名称
- 计划完成
- 实际完成
- 计划完成率
- 完成率
- 监理意见
- 照片链接
- 验收资料

页面内提供模板下载按钮，可直接下载示例表格后填写。

## 技术说明

- `index.html`：页面结构。
- `styles.css`：太空舱中控大屏视觉样式。
- `app.js`：数据管理、Excel 导入、进度计算、3D 沙盘和页面交互逻辑。

数据默认保存在浏览器 `localStorage` 中，点击“恢复示例”可恢复演示数据。
